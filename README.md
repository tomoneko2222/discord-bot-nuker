# discord bot nuker

### Getting Started
- 1.Run`setup.bat`
- 2.Insert token into `bottken.txt`
- 3.Run`main.bat`

### Multiple tokens supported(N/A)
- DMspam        (A)
- Channeldelete (N)
- CreateChannel (N)
- nuker         (A)
- roleDelete    (N)
- roleCreate    (N)
- allkick       (N)
- allban        (N)
- unbanall      (N)
